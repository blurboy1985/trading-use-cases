🛑 Read This First

  • Paper trade for at least 3 months before live deployment
  • Start with read-only API keys, graduate to trading permissions only after testing
  • Implement kill switches - Telegram command to halt all trading
  • Set position limits - max 2-5% per position, max daily loss limits
  • Never give withdrawal permissions to API keys

Broker Comparison

Alpaca (Recommended for US)

Zero commission, excellent paper trading, simple API. Stocks + crypto. No options.

Interactive Brokers (Global)

Full API, global markets, all asset classes. Steep learning curve. Best for serious traders.

Binance (Crypto)

Largest crypto exchange. Spot, futures, options. NOT available to US users. Secure your API keys!
